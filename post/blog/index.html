Hugo

1. 安装Hugo

看看能不能显示？？？

  1. ubantu系统 下载Hugo官方下载
# 下载压缩包
wget https://github.com/gohugoio/hugo/releases/download/v0.81.0/hugo_0.81.0_Linux-64bit.tar.gz


# 解压
tar -xzvf hugo_0.81.0_Linux-64bit.tar.gz 
sudo cp ./hugo /usr/local/bin
  1. 检查是否安装成功
hugo version
  1. 创建blog站点
hugo new  site myblog 
#myblog是项目名
  1. myblog下的目录如下
myblog
├── archetypes
│   └── default.md
├── assets
├── config.toml   # 博客站点的配置文件
├── content	      # 博客文章所在的目录	
├── data
├── layouts		  # 网站布局
├── public		  
├── static		  # 一些静态内容
└── themes		  # 博客主题
  1. 安装主题

官方主题下载：hugo主题

在myblog目录，将要下载的主题克隆到themes目录下

# 切到自己的站点
cd myblog 
# 初始化git
git  init
# clone 到themes/stack下，stack是 为主题hugo-theme-stack起的名称
git clone https://github.com/CaiJimmy/hugo-theme-stack.git   ./themes/stack
git clone https://github.com/CaiJimmy/hugo-theme-stack.git   themes/stack
  1. 将主题配置到配置文件中
echo 'theme="book" '>> config.toml

# 或者直接在config.toml下加入  theme="stack“
  1. 新建文章
hugo new post/blog.md

#blog.md内容
title: "Blog"
date: 2022-12-07T23:23:48+08:00
draft: true

生成的 md 文件在myblog/context/post目录下

  1. 启用hugo服务器
hugo server 

# 选项
-t  选定的主题  # 指定主题

启用草稿的情况下，启用服务器要使用以下命令：

hugo server -D
  1. 写一篇文章
